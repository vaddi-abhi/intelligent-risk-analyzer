# Intelligent Vulnerability & License Risk Analyzer

## Overview
Analyzes software packages to detect vulnerabilities, licenses, and compute risk scores.

## Features
- Dependency Analysis
- Vulnerability Detection using OSV API
- License Detection using PyPI API
- Risk Score Calculation
- Risk Categorization
- Report Generation
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Requests
- OSV API
- PyPI API
- ReportLab

## Installation

Clone Repository

git clone https://github.com/vaddi-abhi/intelligent-risk-analyzer.git

Move into project

cd intelligent-risk-analyzer

Install dependencies

pip install -r requirements.txt

Run application

streamlit run app.py
